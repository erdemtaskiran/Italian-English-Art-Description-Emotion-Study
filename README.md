# Italian-English-Art-Description-Emotion-Study
Emotion analysis of art descriptions using the EmoAtlas library. This Jupyter notebook analyzes Italian text feedback from museum visitors, categorizing emotions (anger, trust, surprise, disgust, joy, sadness, fear, anticipation) using natural language processing.
**Original Source:**
This analysis uses the EmoAtlas library developed by Massimo Stella:
https://github.com/MassimoStel/emoatlas

**Features:**
- Italian language text emotion analysis
- Z-score based emotion scoring
- Processing of museum visitor feedback
- Visualization of emotional patterns in cultural contexts

**Requirements:**
- Python 3.x
- EmoAtlas library
- spaCy, NLTK, NetworkX

**Usage:**
The notebook demonstrates how to apply emotion analysis to real-world museum visitor comments, providing insights into visitor experiences and emotional responses to cultural exhibits.
